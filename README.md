# 项目简介

本项目为基于深度学习YOLOv12的花生种子霉变识别检测系统。主要功能模块包括使用YOLOv12算法对花生种子图像进行实时处理，识别霉变情况，并提供可视化UI界面。系统集成了登录注册功能，保障用户数据安全。技术上，采用YOLO数据集进行模型训练，实现了高效准确的霉变检测，同时，项目源码开放，方便研究者进一步优化与拓展。


## 核心功能
- 使用YOLOv12算法对花生种子图像进行实时处理
- 识别花生种子霉变情况
- 提供可视化UI界面
- 集成登录注册功能，保障用户数据安全
- 项目源码开放，便于进一步研究与开发

## 适用场景
- 农产品质量检测领域
- 农业科研机构
- 农产品加工企业
- 种子质量检测中心
- 农业自动化研究

## 技术特点
- 基于YOLOv12深度学习算法
- 使用YOLO数据集进行模型训练
- 实现高效准确的霉变检测
- 提供完整的项目源码
- 支持用户登录与注册功能
